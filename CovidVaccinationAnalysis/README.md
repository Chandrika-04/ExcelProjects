💉 COVID-19 Vaccination Analysis — Microsoft Excel
An end-to-end public health data analysis project on real-world global COVID-19 vaccination data across 10 countries — from raw data import to a fully interactive Excel dashboard.
✨ Key Features

Clean Data — imported, formatted, and cleaned 86,000+ rows via Power Query with missing value handling
Calculated Columns — Year, Month, Fully Vaccinated %, Dose Gap, and Vaccination Level (Low / Medium / High)
PivotTables — 4 PivotTables covering country totals, monthly rollout, vaccination levels, and completion rates
Dashboard — 5 KPI cards, 3 charts, Year & Country slicers, all fitting on one protected screen

🧰 Excel Concepts Used
YEAR · MONTH · TEXT · IFS · IFERROR · MAXIFS · SUMIF · AVERAGEIF · COUNTIF · INDEX/MATCH · PivotTables · Slicers · Conditional Formatting · Data Bars
📁 Sheets
Raw_Data · Clean_Data · Calculations · PivotTables · Charts · Dashboard
🛠 Requirements

Microsoft Excel 2016 or later
Dataset: COVID-19 World Vaccination Progress (Kaggle)
