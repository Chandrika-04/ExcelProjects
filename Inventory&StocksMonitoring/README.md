# 📦 Inventory & Stock Monitoring — Microsoft Excel

A fully functional Excel inventory monitoring system built on real supply chain data — tracking stock levels, flagging reorder needs, scoring supplier performance, and visualising demand trends.

## ✨ Key Features
- **Inventory Register** — per-SKU stock status (Critical / Low / Adequate / Overstocked) with reorder point logic
- **Supplier Scorecard** — performance summary per supplier (lead time, defect rate, risk rating)
- **Dashboard** — 5 charts covering stock vs reorder, inventory health, capital by category, supplier risk, and monthly demand
- **Inventory Report** — written management summary with procurement recommendations
- **Power Query** — both datasets imported, cleaned, and shaped without manual copy-paste

## 🧰 Excel Concepts Used
`IF` · `AND` · `OR` · `IFERROR` · `VLOOKUP` · `SUMIF` · `AVERAGEIF` · `COUNTIF` · `COUNTIFS` · `Power Query` · `Conditional Formatting` · `Color Scales` · `PivotTables`

## 📁 Sheets
`Raw Data` · `Transactions` · `PQ Screenshots` · `Inventory Register` · `Supplier Scorecard` · `Dashboard` · `Inventory Report`
