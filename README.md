# 📊 Excel Projects — Data Analytics Portfolio

A collection of hands-on Microsoft Excel projects built on real-world datasets — covering inventory management, financial reporting, and meal planning. Each project demonstrates a full analytics pipeline from raw data to dashboards and written insights.

---

## 📁 Projects

### 1. 📦 [Inventory & Stock Monitoring](./Inventory%20stocks/)
A fully functional inventory monitoring system built on real supply chain data — tracking stock levels, flagging reorder needs, scoring supplier performance, and visualising demand trends.

- **Key Skills:** Power Query · VLOOKUP · SUMIF · Conditional Formatting · PivotTables
- **Dataset:** Supply Chain (Kaggle) · Online Retail (UCI)

---

### 2. 🏪 [SuperStore Financial Report](./superstores/)
A professional financial analytics case study on US retail data — modelling a full P&L report, analysing regional performance, and mapping a strategic FY 2020 project timeline via a Gantt chart.

- **Key Skills:** VLOOKUP · IFERROR · PivotTables · Slicers · Conditional Formatting · Gantt via CF Grid
- **Dataset:** SuperStore US 2015 (Power BI Docs)

---

### 3. 🥗 [Meal Planner & Grocery List](./Meal%20Planner/)
An intelligent meal planning system that auto-generates a weekly grocery list from a recipe database — combining INDEX-MATCH lookups, SUMIF aggregation, and interactive checkboxes for a seamless planning-to-shopping workflow.

- **Key Skills:** INDEX-MATCH · SUMIF · Form Checkboxes · Data Validation · Dynamic Lists · Dashboard Design
- **Dataset:** Custom recipe database (built directly in the workbook)

---

## 🧰 Tools & Technologies
- Microsoft Excel 2016 or later
- Power Query
- PivotTables & PivotCharts
