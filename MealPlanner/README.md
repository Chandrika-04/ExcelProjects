# 🥗 Meal Planner & Grocery List — Microsoft Excel

An intelligent Excel workbook that links your weekly meal plan to an auto-updating grocery list. Change a meal and costs, quantities, and calories update instantly.

## ✨ Key Features
- **RecipeDB** — central recipe & ingredient database (6 recipes, 40 ingredients)
- **MealPlan** — 7-day × 3-meal grid with dropdown selection
- **GroceryList** — auto-aggregated shopping list with interactive checkboxes
- **Dashboard** — KPI cards (weekly cost, calories, avg daily spend) + category bar chart

## 🧰 Excel Concepts Used
`INDEX-MATCH` · `SUMIF` · `COUNTIF` · `Data Validation` · `Form Control Checkboxes` · `Conditional Formatting` · `Named Ranges`

## 🛠 Requirements
- Microsoft Excel 2016 or later
- Developer tab enabled (for checkboxes)
